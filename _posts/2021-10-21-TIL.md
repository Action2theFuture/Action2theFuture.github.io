- 하위 디렉토리 검사 후 해당하는 확장자 찾기
```python
from pathlib import Path

path = "/home"
extension = ".txt"

def today(path, extension):
    return [file_path for file_path in Path().glob('**/*') if file_path.suffix == extension]
```

- 현재 위치의 폴더 개수 
```
ls -l | grep ^d | wc -l
```
- 현재 위치의 파일 개수 
```
ls -l | grep ^- | wc -l
```
- 현재 디렉토리의 하위 파일 개수 
```
find . -type f | wc -l
```
