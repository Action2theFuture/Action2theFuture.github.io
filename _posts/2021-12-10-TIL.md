## Reverse Tunneling check

- Local Server

```bash
$ nc -lvp [localport]                           
```

- Remote Server

```bash
$ nc localhost [remote_port]
```

netcat으로 연결이 되었다

Local server에서 채팅을 치면 Remote Server에도 전달이 된다
