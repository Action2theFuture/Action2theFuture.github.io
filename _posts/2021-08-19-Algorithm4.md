# [구명보트](https://programmers.co.kr/learn/courses/30/lessons/42885)

```python
def solution(people, limit):
    people.sort()
    answer = 0
    index = 0
    index2 = len(people) - 1
    while True:
        answer += 1
        if people[index] + people[index2] <= limit:
            index += 1
        index2 -= 1

        if index > index2:
            break
    return answer
```
