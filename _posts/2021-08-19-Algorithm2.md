# [최댓값과 최솟값](https://programmers.co.kr/learn/courses/30/lessons/12939)

```python
def solution(s):
    s = s.split(" ")
    n = []
    for i in s:
        n.append(int(i))
    answer =  str(min(n)) + " " + str(max(n))
    return answer
```
